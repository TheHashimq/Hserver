# Hserver
Simple web server made with c
### Required :
- gcc (GNU compiler collection);
- strace
- wget

### Running 
````bash
$ git clone https://github.com/thehashimq/Hserver
$ cd Hserver
$ gcc main.c -o main
````

#### Execut with strace

````bash
$ strace ./main

````

#### In Another Teminal 

````bash
$ wget localhost:8080/index.html
````
